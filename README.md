# ZUI 3

## 开发

### 项目安装

执行：

```sh
$ pnpm install
```

### 调试

执行下面的命令打开一个 Web 服务来进入调试模式：

```sh
$ pnpm dev
```

在开发模式下，当组件源码目录

针对特定组件启动开发模式可以使用 `--com` 参数，例如启动按钮组件（`button`）的开发模式：

```sh
$ pnpm dev -- --com button
```

### 打包

执行：

```
$ pnpm build
```
