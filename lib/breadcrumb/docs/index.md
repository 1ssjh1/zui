# 面包屑

## 基本用法

使用类 `.breadcrumb` 来实现面包屑，通常搭配 `<ol>`、`<li>` 元素使用。

<Example>
  <ol class="breadcrumb">
    <li><a href="your/url/">首页</a></li>
    <li><a href="your/url/">Library</a></li>
    <li class="active">Data</li>
  </ol>
</Example>

```html
<ol class="breadcrumb">
  <li><a href="your/url/">首页</a></li>
  <li><a href="your/url/">Library</a></li>
  <li class="active">Data</li>
</ol>
```
