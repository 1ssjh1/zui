# 面包屑

## 基本用法

使用组件类`.breadcrumb`来获得外观 通常搭配`<ol>`,`<li>`元素使用

<Example>
  <ol class="breadcrumb">
    <li><a href="your/url/">Home</a></li>
    <li><a href="your/url/">Library</a></li>
    <li class="active">Data</li>
  </ol>
</Example>

```html
   <ol class="breadcrumb">
     <li><a href="your/url/">Home</a></li>
     <li><a href="your/url/">Library</a></li>
     <li class="active">Data</li>
   </ol>
```
