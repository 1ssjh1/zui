# shrink

使用`shrink`应用CSS`flex-shrinl`属性设置Flex容器中元素是否缩小。

<Example class="w-24 flex gap-3">
  <div class="bg-primary w-16 h-16 shrink"></div>
  <div class="bg-primary w-16 h-16 shrink-0"></div>
</Example>
