# grow

使用`grow`应用CSS`flex-grow`属性设置Flex容器中元素是否能放大。

<Example class="flex gap-3 w-full">
  <div class="bg-primary w-16 h-16 grow-0"></div>
  <div class="bg-primary w-16 h-16 grow"></div>
</Example>
