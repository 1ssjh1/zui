# 容器

`container` 类设置一个元素的 `max-width` 来匹配当前断点的 `min-width`。如果您想为一组固定的屏幕尺寸设计，而不是试图适应一个完全流动的视窗，这很有用。

<Example class="p-0">
  <table class="table">
    <thead>
      <tr>
        <th>工具类</th>
        <th>断点</th>
        <th>属性</th>
      </tr>
    </thead>
    <tbody>
      <tr v-for="item in containerJson">
        <td>{{item.name}}</td>
        <td>{{item.breakpoint}}</td>
        <td><code>{{item.desc}}</code></td>
      </tr>
    </tbody>
   </table>
</Example>

## 使用方法

```html
<div class="container">
  <!-- ... -->
</div>
```

::: tip 提示
请注意，容器不会自动居中，也没有任何内置的水平方向的内边距。
:::
