# 导航

## 基本用法

使用组件类`.nav`来获得导航外观 通常搭配`<ul>`,`<li>` 标签来使用

```html:example
<ul class="nav">
  <li class="active"><a href="">首页</a></li>
  <li><a href="">动态 </a></li>
  <li><a href="">项目 </a></li>
</ul>
```

### -default

```html:example
 <ul class="nav -default">
   <li class="active"><a href="">首页</a></li>
   <li><a href="###">动态 </a></li>
   <li><a href="">项目 </a></li>
 </ul>
```


### -simple

```html:example
<ul class="nav -simple">
  <li class="active"><a href="">首页</a></li>
  <li><a href="">动态 </a></li>
  <li><a href="">项目 </a></li>
</ul>
```
