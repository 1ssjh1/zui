# 导航

## 基本用法

使用组件类`.nav`来获得导航外观 通常搭配`<ul>`,`<li>` 标签来使用

```html:example
<ul class="nav">
  <li><a href="###" class="active">首页</a></li>
  <li><a href="###">动态 </a></li>
  <li><a href="###">项目 </a></li>
</ul>
```

### 简单导航

```html:example
<ul class="nav nav-simple">
  <li><a href="###" class="active">首页</a></li>
  <li><a href="###">动态 </a></li>
  <li><a href="###">项目 </a></li>
</ul>
```

## 主要导航

```html:example

  <li><a href="###">动态 </a></li>
  <li><a href="###">项目 </a></li>
```

## 次要导航

## 标签页导航

 ```html:example

   <li><a href="###">首页</a></li>
   <li><a class="active" href="###">动态 </a></li>
   <li><a href="###">项目 </a></li>
 </ul>
 ```
