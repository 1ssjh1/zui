# ajaxForm

## 校验必填项

校验表单的必填项，并将错误提示显示在表单对应位置下。

## 监听表单变化

监听表单内的数据变化，点击提交按钮前若关闭当前页面进行对应提示。

[示例待定]


## 成功的回调

用户可将处理成功的回调添加到 `success` 函数里。表单成功提交后进行对应逻辑处理。

[示例待定]


## 失败的回调

用户可将处理失败的回调添加到 `fail` 函数里。表单失败提交后进行对应逻辑处理。

[示例待定]


## 完成的回调

无论成功失败，用户可将完成的回调添加到 `complete` 函数里。表单失败提交后进行对应逻辑处理。
