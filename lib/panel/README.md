# 面板

## panel

```html:example: flex gap-3
<div class="panel">
  <div class="panel-body">
    默认的面板所做的只是提供基本的边界和内部，来包含内容。
  </div>
</div>
```

## with panel-heading

```html:example: flex gap-3
<div class="panel">
  <div class="panel-heading">面板标题</div>
  <div class="panel-body">面板内容</div>
</div>
```

## width panel-footer

```html:example: flex gap-3
<div class="panel">
  <div class="panel-body">面板内容</div>
  <div class="panel-footer">面板脚注</div>
</div>
```
